# DESCRIÇÃO DO PROJETO

PROJETO SEGUIDOR SOLAR COM MONITORAMENTO WEB

O objetivo do projeto do seguidor solar com monitoramento web é demonstrar de forma didática como funciona um seguidor solar (ou tracker ou rastreador solar) e como ela é importante para a geração de energia elétrica através do sol, considerando a aplicabilidade dela a eficiencia energética.
Para isso, usamos os seguintes componentes para tal projeto:
1 ESP32 (processamento e controle do código-fonte);
1 TELA OLED 1306 I2C
2 LDRs
1 SERVOMOTOR PARA MOVIMENTO DO SUPORTE DE ONDE MOVIMENTARÁ OS MÓDULOS
1 MÓDULO FOTOVOLTAICO
1 SENSOR DE CORRENTE OU DE TENSÃO (OPCIONAL)
1 PROTOBOARD
